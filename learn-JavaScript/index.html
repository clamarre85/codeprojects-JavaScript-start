JavaScript Functions and Events

A JavaScript function is a block of JavaScript code, that can be executed when "called" for.

For example, a function can be called when an event occurs, like when the user clicks a button.

External JavaScript

Your JavaScript can be placed in files that have the extension .js, and linked to your page.

Useful when the same code is used in many different web pages.

External JavaScript Advantages:
    • Separates HTML and Code
    • Makes HTML and JavaScript easier to read and maintain
    • Cached JavaScript files can speed up page loads

To add several script files to one page - use several script tags:
script src = " myScript1.js "
script src = " myScript2.js "

External References

An external script can be refrenced in 3 different ways:
    • Full URL (full web address)
    • File Path (like linking .js)
    • No Path
An example that uses a Full URL to link to myScript.js:

↳ script src = "https://www.w3schools.com/js/myScript.js"


An example that uses a File Path to link to myScript.js:

↳ script src = "/ js / myScript.js"


An example that uses a No Path to link to myScript.js:

↳ script src = "myScript.js"

Using document.write( )

*** Using document.write( ) after an HTML document is loaded, will delete all existing HTML!

Never call document.write after the document has finished loading because it will overwrite the entire document.
The document.write( ) method should only be used for testing.

Using window.alert ( )

You can use an alert box to display data.
Specifying the window keyword is optional because in JavaScript the window object is the global scope object.
JavaScript is an "Object-Based" Programming, Scripting Language.
This means that variables, properties, and default methods belong to the window object.

Semicolons ;

JavaScript statements are separated by semicolons.
Add a semicolon at the end of each executable statement. It is not required, but recommended.
When separated by semicolons, multiple statements on one line are allowed.


JavaScript White Space

JavaScript ignores multiple spaces.
You can add white space to your script to make it more readable but they are equivalent:

let person = "Amber"
let person="Amber"

A good practice is to put spaces around operators ( = + - * / ):

let x = y + z;

JavaScript Syntax

JavaScript Syntax is the set of rules, how JavaScript programs are constructed:
//How to create variables:
var x;
let y;

//How to use variables:
x = 5 ;
y = 6 ;
let z = x + y;

JavaScript Values

The JavaScript syntax defines two types of values:
    • Fixed Values
    • Variable Values

Fixed values are called Literals.

Variable values are called Variables.

JavaScript Literals

The two most important syntax rules for fixed values are:
  1. Numbers are written with or without decimals.
  2. Strings are text, written within double or single quotes.

JavaScript Variables

In a programming language, variables are used to store data values.

JavaScript uses the keywords var, let, and const to declare variables.

An = sign is used to assign values to variables.

In this example, f is defined as a variable. Then f is given the value of 6:

let f;

f = 6 ;


Written in .js file it would look something like this to produce a calculation behind the scenes:

let f;
f = 6;
document.getElementById("value").innerHTML = f;
